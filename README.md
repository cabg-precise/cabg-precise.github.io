# CABG-PRECISE Risk Score
Preoperative risk score designed to predict in-hospital mortality following coronary artery bypass graft.

This tool was developed and validated at Hospital General Universitario Gregorio Marañón (Madrid, Spain) using a cohort of 2,101 consecutive patients, featuring a high discriminatory capacity (AUC 0.810).
